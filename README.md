## 📌 Project Objective

Esports has evolved from small LAN tournaments into a global industry with millions of viewers, professional organizations, sponsorships, and multi-million-dollar prize pools.

This project analyzes different aspects of the Counter-Strike esports ecosystem to answer questions such as:

- How has the prize pool changed over time?
- Which teams have dominated the competitive scene?
- Which players have earned the most prize money?
- How has tournament viewership evolved?
- What trends can be observed across tournaments and teams?

Although this dashboard focuses on Counter-Strike 2, it represents the broader evolution of the esports industry through data.

---

## 📂 Dataset Used

The dashboard combines five datasets:

- Tournament Results
- Player Statistics
- Team Rankings
- Match Viewership
- Game Economics

These datasets were cleaned, transformed, and integrated into a single Power BI data model.

## ⚙️ Project Workflow

### 🧹 Data Cleaning

- Removed duplicate records
- Handled missing values
- Standardized column names
- Corrected data types
- Cleaned inconsistent values

### 🔄 Data Transformation (Power Query)

- Imported multiple CSV datasets
- Merged and transformed tables
- Created calculated columns
- Optimized data for reporting

### 🏗 Data Modeling

Created relationships between:

- Players
- Teams
- Tournaments
- Match Viewership
- Prize Pool Information

to build an optimized relational data model.

### 📐 DAX Measures

Developed measures for:

- Total Prize Pool
- Total Players
- Total Teams
- Tournament Count
- Average Prize Pool
- Player Earnings
- Team Wins
- Peak Viewership
- Dynamic KPI Cards

### 📊 Dashboard Features

- KPI Cards
- Tournament Analytics
- Top Players by Earnings
- Top Teams by Wins
- Prize Pool Trends
- Viewership Analysis
- Interactive Filters
- Tournament Results Table

## 📈 Key Insights

- 💰 Over **$35 Million** in total prize money.
- 👀 Peak tournament viewership exceeded **2 Million**.
- 🌍 Teams from multiple countries compete on the global stage.
- 🏆 A few organizations consistently dominate tournaments.
- 📈 Prize pools have steadily increased, reflecting the rapid growth of esports.
- 🎮 Counter-Strike showcases how gaming has evolved into a professional global industry.
